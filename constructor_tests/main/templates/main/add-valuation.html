Добавление шкалы оценки теста

Количество оценок не ограниченно. Каждый вариант оценки добавляется по отдельности.

Для добавления результата нужно заполнить каждое поле формы представленной ниже, где:

  • "Уровень результата" - название уровня (к примеру: Низкий / Средний / Высокий).

  • "Верхняя граница" - верхнее значение границы результата (в баллах), соответствующее уровню оценки. Нижняя граница задается автоматически: для первого уровня она равна "0", для последующий, она равна значению верхней границы предыдущего уровня

  • "Интерпретация оценки" - описание того, что означает набранный уровень баллов.
